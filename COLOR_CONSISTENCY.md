## 🎨 配色方案总览

### 主色调（蓝色系）
```
深海蓝 #1E3A5F - 导航、Logo、主按钮
天际蓝 #3B82F6 - 按钮、链接、进度条
晴空蓝 #0EA5E9 - 次级按钮
淡蓝   #DBEAFE - 背景
```

### 强调色
```
琥珀金 #F59E0F - 支线任务、规划页面
森林绿 #10B981 - 完成状态、日报
暖橙   #F97316 - 阻塞、警告
砖红   #EF4444 - 逾期、删除
```

### 优先级（渐进暖色）
```
P0 #DC2626 深红 → P1 #EA580C 橙红 → P2 #EAB308 金黄 → P3 #9CA3AF 灰色
```

### 中性色
```
背景白   #F8FAFC
卡片白   #FFFFFF
边框灰   #E2E8F0
分割线   #CBD5E1
次要文字 #94A3B8
主要文字 #64748B
标题文字 #1E293B
```
